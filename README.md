1. Import LevelPlay package using this link: https://developers.is.com/ironsource-mobile/unity/unity-plugin/#step-1

Use UPM to install package.

2. Install External Dependency Manager.

3. Add this package in `Unity Package Manager` using the following URL:

```
https://github.com/MirraSDK/SDK5-LevelPlay-API.git
```

4. Go to MirraSDK5 Toolkit, select configuration that you plan to build (e.g. Google Play), open Ads section, select LevelPlayAds for Ads provider.

5. Enter App key and Ad unit ids.
